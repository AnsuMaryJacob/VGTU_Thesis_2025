'--- Interconnections
Chatbot --> DataLake : store chat logs
DataLake --> Anonymizer : provide cleaned text
Anonymizer --> API : send anonymized data / requirements inputs

Portal --> API : submit queries / documents
API --> Embedder : compute embeddings
Embedder --> Index : store/retrieve embeddings
API --> RAG : retrieve relevant context
RAG --> LLM : send prompt (query + context)
LLM --> API : return generated requirement text
API --> Classifier : label FR/NFR
API --> Validator : validate requirement
Validator --> Repo : save validated requirement
API --> Portal : show results
Portal --> Trainer : collect feedback
Trainer --> Index : update embeddings
Trainer --> LLM : refine prompt template
Repo --> Audit : report compliance data
